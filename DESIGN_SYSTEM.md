## 📋 项目概述

## 🔳 组件样式

### 卡片 (Cards)

```css
背景: var(--bg-card)
边框: 1px solid var(--border)
圆角: .75rem
内边距: 1.75rem
阴影: var(--shadow-sm)
过渡: .22s cubic-bezier(0.4, 0, 0.2, 1)
```

**悬停效果**:
- 升起: `translateY(-4px)`
- 阴影增强: `var(--shadow-card)`
- 边框变金: `var(--accent)`

### 按钮 (Buttons)

#### 主按钮 (btn-primary)
```css
背景: #d4af37 (金色)
文字: #0a0e27 (深黑)
阴影: 0 4px 12px rgba(212, 175, 55, 0.25)
```

**悬停**:
- 背景: #e5c158 (亮金)
- 阴影: 0 6px 20px rgba(212, 175, 55, 0.35)

#### 轮廓按钮 (btn-outline)
```css
边框: 1.5px solid #d4af37
文字: #d4af37
```

**悬停**:
- 背景: #2a2416 (浅金背景)

### 表单控件 (Forms)

```css
背景: #0f131e (暗输入)
边框: 1.5px solid #1f1a2a
文字: #e8e3d8
```

**焦点**:
- 边框: #d4af37 (金色)
- 光环: 0 0 0 4px rgba(212, 175, 55, .12)

### 徽章 (Badges)

所有徽章使用**玻璃态** (Glassmorphism) 效果:
- 背景: 半透明颜色 (15% 不透明度)
- 边框: 半透明深色边框 (30% 不透明度)
- 圆角: `6px`

**颜色变体**:
- 开放: 金色系
- 已分配: 紫蓝系
- 进行中: 青色系
- 待确认: 橙色系
- 已完成: 绿色系

### 导航栏 (Navbar)

```css
背景: #0a0e27
边框-底: 1px solid #2a2530
阴影: 0 4px 20px rgba(0, 0, 0, .6)
背景滤镜: blur(8px)
```

### 模态框 (Modals)

```css
背景: rgba(0, 0, 0, .65)
背景滤镜: blur(4px)
```

**内容**:
```css
背景: #141829
边框: 1px solid #2a2530
圆角: .875rem
内边距: 2rem
阴影: 0 20px 60px rgba(0, 0, 0, .8)
```

## ✨ 动画和过渡

### 过渡时间

```css
--transition: .22s cubic-bezier(0.4, 0, 0.2, 1)
```

### 关键动画

#### 淡入 (fadeIn)
```css
@keyframes fadeIn {
    from { opacity: 0; transform: translateY(8px); }
    to   { opacity: 1; transform: translateY(0); }
}
```

#### 通知进入 (toastIn)
```css
@keyframes toastIn {
    from { opacity: 0; transform: translateX(20px); }
    to   { opacity: 1; transform: translateX(0); }
}
```

#### 脉冲发光 (glowPulse)
```css
@keyframes glowPulse {
    0%, 100% { box-shadow: 0 0 12px rgba(212, 175, 55, 0.2); }
    50% { box-shadow: 0 0 20px rgba(212, 175, 55, 0.4); }
}
```

## 🎯 设计原则

1. **极简主义** - 移除视觉混乱，强调内容
2. **高级感** - 使用金色强调和精致的间距
3. **可读性** - 高对比度确保易读性
4. **一致性** - 统一的色彩、排版和间距
5. **交互性** - 清晰的悬停和焦点状态
6. **响应性** - 完美适配所有设备尺寸

## 📱 响应式设计

### 断点

- **大屏幕**: `768px` 以上
- **中等屏幕**: `640px` - `768px`
- **小屏幕**: `640px` 以下

### 调整

| 设备 | 网格 | 调整 |
|------|------|------|
| 台式机 | 多列 | 全栏宽 |
| 平板 | 2-3列 | 调整卡片 |
| 手机 | 1列 | 堆叠布局 |

## 🔧 开发指南

### 使用 CSS 变量

```css
/* 背景 */
background: var(--bg-card);

/* 文本 */
color: var(--text-primary);

/* 强调 */
color: var(--accent);

/* 阴影 */
box-shadow: var(--shadow-card);

/* 过渡 */
transition: color var(--transition);
```

### 添加新颜色

在 `:root` 中添加新变量:

```css
:root {
    --my-new-color: #ffffff;
}
```

### 响应式查询

```css
@media (max-width: 768px) {
    /* 中等屏幕样式 */
}

@media (max-width: 640px) {
    /* 小屏幕样式 */
}
```
